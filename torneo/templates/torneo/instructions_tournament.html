Esta etapa tendrá 5 rondas. En cada ronda usted hará parte de un grupo de cuatro (4) personas. Dos (2) tendrán un Contrato A y dos (2) un Contrato B. Los otros 3 miembros de su grupo irán cambiando ronda tras ronda.

En cada ronda usted tendrá 90 segundos para realizar el mayor número de tareas posibles. El pago por cada tarea completada dependerá del contrato al que fue asignado en la ronda anterior:

  • Contrato A: recibe $2.000 por tarea correcta.
  • Contrato B: recibe $1.000 por tarea correcta.

Al final de cada ronda se reasignarán los Contratos A y B según el número de tareas completadas respecto a los otros miembros del grupo.

Probabilidad de tener un Contrato A en la siguiente ronda:
  • Primer puesto de los jugadores con Contrato A: {% if discrimination == 0 %} 50% {% else %} 100% {% endif %}
  • Segundo puesto de los jugadores con Contrato A: {% if discrimination == 0 %} 50% {% else %} Repechaje {% endif %}
  • Primer puesto jugadores con Contrato B: {% if discrimination == 0 %} 50% {% else %} Repechaje {% endif %}
  • Segundo puesto jugadores con Contrato B: {% if discrimination == 0 %} 50% {% else %} 0% {% endif %}

Nota: Si los dos participantes con el mismo contrato tienen el mismo número de tareas completas, el primer puesto del contrato se define al azar.

En otras palabras, {% if discrimination == 0 %} la probabilidad de ser cambiar de contrato será del 50% para todos los jugadores. {% else %} El “ganador” con el Contrato A y el “perdedor” del Contrato B van a mantener sus contratos. Para las otras dos personas del grupo, es posible que haya un cambio en el contrato.

La probabilidad de ser reasignado dependerá del número relativo de tareas realizadas, en algo que llamaremos “repechaje”.

¿En qué consiste el repechaje?

Hay dos participantes en el repechaje. Entre más tareas correctas haga, y menos haga el otro participante, más chance de quedarse con el Contrato A.

{% if discrimination == 1 %}

Usted ganará el Contrato A si hace más tareas completas que el otro participante en repechaje.

{% else %}

Su probabilidad de ganar el Contrato A es igual a sus tareas completas, dividido en las tareas completas de los dos participantes en repechaje:

Probabilidad del Contrato A =
Mis tareas / Mis tareas+Tareas del otro
